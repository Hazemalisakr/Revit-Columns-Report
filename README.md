Revit Columns Report Exporter
A powerful automation tool for Revit that extracts structural and architectural column data and exports it to a formatted Excel report. Includes site coordinates (Easting/Northing) and volumetric data.

🚀 Key Features
Comprehensive Data Extraction: Exports Easting, Northing, Base Offset, Top Offset, Height, and Volume.
Excel Integration: Uses ClosedXML to generate clean, professional .xlsx reports.
Filtered Selection: Automatically distinguishes between Structural and Architectural columns.
Custom UI: WinForms-based dialog for choosing the export path and configuring options.
KAITECH Ribbon Tab: Seamlessly integrated into the Revit interface.
